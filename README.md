# AWS Serverless API Project

This project demonstrates a simple serverless REST API built using AWS services.
It shows how to expose a backend endpoint with AWS Lambda and Amazon API Gateway.

---

## Architecture

- AWS Lambda (Python)
- Amazon API Gateway (HTTP API)
- IAM Role for secure execution

---

## Features

- Serverless API endpoint
- GET route returning a JSON response
- Automatic deployment using API Gateway
- Scalable and cost-efficient design
